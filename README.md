# SQL Data Analytics Project 📊

A comprehensive collection of production-grade SQL scripts engineered for advanced data exploration, business analytics, and performance reporting. These scripts demonstrate how to turn raw transactional data into structured, actionable business intelligence using optimized T-SQL practices.

## 🏢 Business Scenarios Covered
- **Product Performance Tracking:** Aggregates critical product-level metrics including total orders, revenue generation, unique customer counts, and product lifespans to classify items into behavioral segments (High, Mid, or Low performers).
- **Time-Series & Trend Analysis:** Evaluates core metrics and tracking shifts sequentially over specific periods to monitor organizational trajectory.
- **Advanced Metrics:** Implementations of rolling/cumulative sums, moving averages, data segmentation techniques, and part-to-whole analyses.

## 🛠️ Technical Concepts Demonstrated
- **Window Functions:** Leveraging `SUM() OVER()`, `ROW_NUMBER()`, and analytical framing to compute running metrics without heavy self-joins.
- **CTEs (Common Table Expressions):** Structuring complex logic into clean, readable, and modular execution blocks.
- **Aggregation & Filtering:** Utilizing robust data segmentation strategies to separate baseline behavior from statistical anomalies.

## 📂 Repository Structure
- `/Scripts`: Contains production `.sql` files organized by core analytical functions.
